영역을 나누는 형식(성질)

  
block 형식


    - 공간을 수직 분할
    -> 수직으로 되어있는 화면을 가로로 잘라서 여러 행을 만드는 것

    - 크기를 지정하는 width, height 속성을 사용할 수 있음 (O)

    ex ) p/pre(문단 나누기), h1~h6(제목 영역 나누기), hr(수평선 그어 나누기), div(영역 나누기)
      
  
inline 형식


    - 공간을 수평 분할
    -> 수평으로 되어있는 화면을 새로로 잘라서 여러 열(컬럼)을 만드는 것

    - 크기를 지정하는 width, height 속성을 사용할 수 없음 (X)

    ex ) strong/b (굵은 글씨) em/i (기울임꼴) mark(형광펜), span(한 줄을 나누는 용도)

block(div) / inline(span) 차이점 2 : 영역 지정 방식

  • block : 사각형 박스 형태로 영역을 지정
  • inline : 내부에 작성된 내용(content) 단위로 영역을 지정
